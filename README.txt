Packer Template for building Instant Veins.

This template may need a number of files to be present in the files/ subdirectory.
See files/SHA1SUMS for a full list.

Type ./configure to check prerequisites and change default values. For more information show at the help section of the ./configure.
Type make to start the build process.
The virtual appliance (as .ova) will be created in the output/ subdirectory.
